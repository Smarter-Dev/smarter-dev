Connection Pool Management

Use this page to manage database connections. If you're experiencing connection pool issues, you can dispose of all connections to reset the pool.

Warning: Disposing connections will close all active database connections. This should only be done when necessary, such as when experiencing connection pool timeouts.

Connection Pool Information

The database connection pool is configured with the following settings:

  • Pool Size: 5 connections
  • Max Overflow: 10 connections
  • Pool Timeout: 30 seconds
  • Pool Recycle: 1800 seconds (30 minutes)
  • Pool Pre-Ping: Enabled

If you're experiencing frequent connection pool timeouts, consider reviewing your application's database usage patterns to identify and fix connection leaks.
