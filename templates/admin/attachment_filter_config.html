    Filter Status

    Enable or disable the attachment filter
    When enabled, the bot will only allow attachments with approved file extensions. All other file types will be blocked.

    Allowed File Extensions

    Enter file extensions to allow (one per line or comma-separated). All others will be blocked.
    Enter file extensions with or without the leading dot. Examples: .png, jpg, .pdf

    Common safe extensions

    Images: .png .jpg .jpeg .gif .webp .svg
    Documents: .pdf .txt .doc .docx .xls .xlsx
    Media: .mp4 .mp3 .wav .mov .webm
    Archives: .zip .rar .7z

    Action on Non-Allowed Files

    What to do when a disallowed file type is detected
    Note: Users with the "Manage Messages" permission are exempt from message deletion. They will only receive a warning instead.

    Warning Message

    Customize the message sent when a disallowed file is detected
    Available placeholders: {user} (mentions the user), {extension} (the file extension), {filename} (the filename)
    Default message:
    @user, your message was removed because the file type (.exe) is not allowed. Please use an approved file format.
    Example custom message:
    {user} - The file "{filename}" ({extension}) is not an approved file type. Allowed formats: images, PDFs, and text files.
